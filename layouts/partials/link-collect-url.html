{{- $ctx := .ctx -}} {{- $slot := lower (printf "%v" (.slot | default "")) -}} {{- $url := printf "%v" (.url | default "") -}} {{- $allowedSlots := slice "nav" "sort" "desc" "post" -}} {{- if and $ctx (in $allowedSlots $slot) $url -}} {{- $candidates := $ctx.Scratch.Get "linkCandidates" | default (dict "nav" (slice) "sort" (slice) "desc" (slice) "post" (slice)) -}} {{- $slotUrls := index $candidates $slot | default (slice) -}} {{- $slotUrls = $slotUrls | append $url -}} {{- $ctx.Scratch.Set "linkCandidates" (merge $candidates (dict $slot $slotUrls)) -}} {{- end -}} {{- return "" -}}
